# 常见问题

## 一：软件提示入库成功，库里面没游戏

进行如下自查：

0. 请保证你已经使用过修复工具，如果没有请前往 [3a.lol](https://3a.lol) 下载

1. Steam tools 是否安装，并且是否开启解锁模式

2. Steam 是否开启了测试版本，如果开了关掉

3. 检查 steam tools 是否开启「禁止游戏更新」，如果开了关掉

4. 检查是否有其他入库工具冲突（例如 GreenLuma、网上卖的垃圾套壳入库工具、假 CDK 等），如果有全部清理。如果依然不行，将 Steam 全部重装

5. 如果以上皆没有解决，重装 Steam tools

6. 如果以上都没有解决，重装 Steam

7. 以上方法皆没有解决，说明 Steam tools 服务器挂掉了，等着

## 三：常见错误及解决方案

### 无互联网连接

- 检查网络连接
- lua/清单/错误
- 单纯网不好，多尝试几次
- 清缓存

### 内容仍处于加密状态

- **原因**: lua 文件过期
- **解决**: 更新 lua 文件

### 内容不可用

- **原因**: 清单/lua 错误或过期
- **解决**: 更新清单或 lua

### 下载 0KB

- **原因**: lua/清单无效或错误
- **解决**: 重新获取有效的 lua 或清单

### 安装路径无效

- **原因**: lua 中缺少 token
- **解决**: 检查 lua 配置

### 6-.../54-... 报错

- **原因**: lua 中缺少 ticket
- **解决**: 更新 lua 文件

### 无许可

- **原因**: 解锁工具（Steam tools 等）未正常工作
- **解决**: 参照「入库成功但库里没游戏」的方法解决

---

## 💡 提示

遇到如上问题，如果你的游戏有 DLC，可以取消几个 DLC 看看能否下载。如果依然不行，请重新找其他库入库。

遇到无法解决的问题，请前往社区（如 caigamer.cn）寻找游戏清单。
