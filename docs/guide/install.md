二进制文件：

### 方式2：

 从上方开源地址下载最新版本的源代码
- 确保已经安装了 Python 3.11 或更高版本
- 运行 `fluent_install.bat` 安装必要的依赖
- 等待安装完成
- 运行 `python main.py` 启动程序
- 或者运行 `run_fluent.bat` 快速启动
